# Samsmrti

A modern spaced-repetition desktop app built with **Tauri**, **React**, and **SQLite**.

## Documentation

- **[QUICKSTART.md](./QUICKSTART.md)** — install, run, profiles, export/import, features
- **[DEVELOPMENT.md](./DEVELOPMENT.md)** — dev setup, tests, **bundling for distribution**

```bash
npm install
npm run tauri dev
```

## Highlights

- FSRS-style scheduling with **per-profile** queues on shared decks
- Deck hierarchy, cloze/reversed note types, Anki/Mochi import
- Knowledge graph (entities, triples) and entity-linked review
- **Content export/import** (JSON) and **full backup/restore** (binary zip)

In-app documentation: open **Help** after launching.
